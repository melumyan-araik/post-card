# 🎉 Конструктор поздравительных открыток

Веб-приложение для удобного создания персональных открыток. Основное предназначение — возможность красиво дарить цифровые сертификаты с персонализированными поздравлениями.


## ✨ Возможности

- **Редактор в реальном времени** - изменения отображаются мгновенно
- **Персонализация** - настройка имен, пожеланий, личных сообщений
- **Слайдер изображений** - загрузка и управление фотографиями
- **QR-коды и сертификаты** - добавление дополнительных элементов
- **Современный дизайн** - адаптивный интерфейс с анимациями
- **Сохранение и экспорт** - возможность сохранить настройки

## 🚀 Как использовать

### 1. Запуск конструктора
Откройте файл `postcard/constructor.html` в браузере.

### 2. Настройка открытки
В левой панели заполните поля:
- **Заголовок открытки** - основной заголовок
- **Событие** - повод для поздравления
- **Имя получателя** - кому адресовано поздравление
- **Имя отправителя** - кто поздравляет
- **Поздравления** - основные пожелания
- **Личное сообщение** - персональное обращение
- **Фотографии** - изображения для слайдера
- **QR-код** - дополнительный элемент

### 3. Предварительный просмотр
В правой панели в реальном времени отображается, как будет выглядеть открытка.

### 4. Сохранение
Нажмите кнопку "💾 Сохранить открытку" для:
- Сохранения в localStorage браузера
- Скачивания файла `setings.js` с настройками

### 5. Просмотр результата
Откройте `postcard/index.html` для просмотра готовой открытки.

## 🛠️ Технические особенности

## 💾 Сохранение данных

### Формат данных
```javascript
{
  "title": "Заголовок открытки",
  "event": "Событие",
  "recipientName": "Имя получателя",
  "wishes": ["Пожелание 1", "Пожелание 2"],
  "senderName": "Имя отправителя",
  "personalMessage": "Личное сообщение",
  "images": ["url1", "url2", ...],
  "qrCode": "url",
  "pushPins": [{}, {}, {}, {}]
}
```

### Места хранения
1. **LocalStorage** - для быстрого доступа
2. **setings.js** - для постоянного хранения
3. **Файл экспорта** - для переноса настроек

## 🔧 Настройка

### Изменение стилей
Отредактируйте файлы в папке `postcard/css/`:
- `style.css` - основные стили
- `partner_card.css` - стили открытки

### Добавление функций
Используйте файл `postcard/script/constructor_integration.js` для расширения функциональности.

## 📞 Поддержка

При возникновении проблем:
1. Проверьте консоль браузера (F12)
2. Убедитесь, что все файлы загружены
3. Проверьте поддержку JavaScript в браузере

# Конструктор поздравительных открыток

Веб-приложение для создания персональных поздравительных открыток с анимацией конверта, слайдером фотографий и QR-кодом. Итог работы — один HTML-файл, который можно разместить на любом хостинге.

## Быстрый старт

```bash
node build.js        # собрать admin.html
open admin.html      # открыть редактор
```

Или запустить через локальный сервер:

```bash
npx serve -p 3456 .
# открыть http://localhost:3456/admin.html
```

## Как пользоваться

1. Открыть `admin.html` в браузере
2. Заполнить поля слева: событие, получатель, пожелания, фото, QR-код
3. Превью справа обновляется автоматически
4. Переключатель **«Анимация»** в тулбаре:
   - **Выкл.** (по умолчанию) — содержимое открытки видно сразу, удобно при редактировании
   - **Вкл.** — показывает конверт с анимацией открытия, как увидит получатель
5. Нажать **«Скачать открытку (HTML)»** — скачается `card.html`
6. Разместить `card.html` на сервере — он полностью автономен, внешние зависимости не нужны

## Что внутри card.html

Экспортированный файл содержит всё необходимое:

- CSS + шрифты GothamPro (base64)
- jQuery, Lodash, Clipboard.js, Cleave.js
- Слайдер и анимация конверта (vanilla JS)
- Загруженные фотографии и QR-код (base64)
- Данные открытки как встроенный JS-объект

Никаких внешних запросов — файл работает даже без интернета.

## Формат данных

```javascript
const jsonData = {
  title:           "Поздравление",
  event:           "День Рождения",
  recipientName:   "Имя получателя",
  wishes:          ["Пожелание 1", "Пожелание 2"],
  senderName:      "Имя отправителя",
  personalMessage: "Личное сообщение",
  images:          ["data:image/jpeg;base64,..."],  // base64
  qrCode:          "data:image/png;base64,...",     // base64
  pushPins:        []
};
```

## Пересборка admin.html

Если изменились файлы в `postcard/` (CSS, JS, шрифты) — пересобери:

```bash
node build.js
```

Скрипт читает исходники, встраивает шрифты как base64 и генерирует новый `admin.html`.

## Стек

- Vanilla HTML/CSS/JS — без фреймворков и сборщиков
- jQuery 2.1.3 для анимаций конверта
- Node.js только для сборки `admin.html` (build.js)

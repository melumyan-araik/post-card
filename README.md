# Конструктор поздравительных открыток

Визуальный редактор для создания персональных поздравительных открыток с анимированным конвертом, слайдером фотографий и QR-кодом. На выходе -- один HTML-файл, полностью автономный и готовый к размещению на любом хостинге.

## Быстрый старт

```bash
node build.js        # собрать admin.html
open admin.html      # открыть в браузере
```

## Возможности

- **4 шаблона оформления** -- Стандарт, Ночной, Золотое Яблоко, Природа
- **Кастомизация цветов** -- акцент, фон карточки, текст, пожелания
- **Конверт** -- 5 анимаций входа, цвет тела и клапанов, своя печать (сургуч)
- **Контент** -- слайдер фотографий (до 10), QR-код / сертификат
- **Экспорт** -- один автономный HTML-файл, работает офлайн

## Как пользоваться

1. Открыть `admin.html` в браузере
2. Выбрать шаблон, настроить цвета и конверт
3. Заполнить поля: событие, получатель, пожелания, фото, QR-код
4. Переключить **Анимация вкл.** для проверки
5. Нажать **Скачать открытку (HTML)**
6. Разместить скачанный `card.html` на сервере

## Система шаблонов

Шаблоны -- JSON-файлы в `templates/`. Каждый определяет цвета, настройки конверта, типографику и лейаут. Цвета инжектируются как CSS-переменные через `:root{}`.

## Пересборка

При изменении исходников в `postcard/` или шаблонов в `templates/`:

```bash
node build.js
```

## Стек

- Vanilla HTML/CSS/JS
- jQuery 2.1.3 для анимаций конверта
- Node.js для сборки `admin.html`
